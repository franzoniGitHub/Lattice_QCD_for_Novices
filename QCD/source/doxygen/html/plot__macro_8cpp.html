Detailed Description

ROOT macro to generate a plot from the QCD_POST.cpp output.

ROOT macro to generate a plot from the QCD_POST.cpp output of the method Metropolis::ComputeRxTWilsonLoops which is stored in the file RXT_potential_plot_file.dat. Various graphical options are implemented and tuned in the code. Type:
$ root plot_macro.cpp
to run and generate the plot "QuarkPotential_root.png".
