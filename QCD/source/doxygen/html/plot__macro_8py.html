Detailed Description

Python macro to generate a plot from the QCD_POST.cpp output.

Python macro to generate a plot from the QCD_POST.cpp output of the method Metropolis::ComputeRxTWilsonLoops which is stored in the file RXT_potential_plot_file.dat. Various graphical options are implemented and tuned in the code. Set the maximum value of r/a (r_max) to be displayed in the PARAMETER section of the script. Type:
$ python plot_macro.py
to run and generate the plot "QuarkPotential_py.png".
