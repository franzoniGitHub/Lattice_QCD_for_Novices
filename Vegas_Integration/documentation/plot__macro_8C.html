Detailed Description

ROOT macro to generate a plot from the VegasHarmOsc.C output.

ROOT macro to generate a plot from the VegasHarmOsc.C output. This macro also performs a fit to the function fitfunction, providing an estimate for the ground energy E_0. Various graphical options are implemented and tuned in the code. Type:
$ ROOT plot_macro.C
to run and generate the plot "plot_ROOT.png".
