Detailed Description

Path class

This class represents a generic lattice configuration. The lattice considered is a 4D space-time lattice, where at each position 4 3x3 complex matrices are located. Periodic boundary conditions are built in thanks to the interface with my4Vector class. Access and set methods are implemented so as to simplify the notations in calculations of functions of the lattice sites.
