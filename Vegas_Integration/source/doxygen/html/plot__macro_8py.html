Detailed Description

Python code to generate a plot from the VegasHarmOsc.cpp output.

Python code to generate a plot from the VegasHarmOsc.cpp output. This macro also performs a fit to the function fitplot_py(), providing an estimate for the ground energy E_0. Various graphical options are implemented and tuned in the code. Type:
$ python plot_macro.py
to run and generate the plot "plot_py.png".
