Detailed Description

Header file to set the adjustable parameters for the experiment code.

Set here the parameters of the system, namely: the 4D grid dimension NCells, the grid spacing a (in fm units), the values of beta, beta_tilde and u0 in the Wilson lagrangian and its improved version, the typical magnitude of a path update epsilon, the number of SU3 matrices to be generated NofSU3, the number of correlated path configurations to skip Ncorr, the number of internal updates on each link inner, the number of path configurations to be sampled Ncf, the boolean option improved to select the action to use and the name of the output file filename.
